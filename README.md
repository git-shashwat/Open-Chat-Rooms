# Open-Chat-Rooms

## A Socket.IO based non remembering Open Chat Room facility.

### Tech. Used
1. NodeJS
2. Express
3. Socket.io

### Features
1. Join Private Rooms
2. Chat 
3. Chat deletes itself as you leave the chat.
